#HW8 DNA Sequence Alignment

## Compile
```Bash
javac DNASequenceAlignment.java
java DNASequenceAlignment

javac DNASequenceAlignmentWithInvalidPairs.java
java DNASequenceAlignmentWithInvalidPairs
```

## Run
Input 2 DNA sequence, for example: 
	T T C G C T 
	A A G G C T

## Exit
Input `exit`  

##Error detection
1. Invalid DNA sequence
2. Empty string
